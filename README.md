# CampusNote Pro 🎓

CampusNote Pro is a modern, academic document-sharing platform designed for Istanbul Arel University students. It simplifies the way students share, discover, and organize study notes while integrating a smart document lifecycle and gamification features.

> **Note:** This repository currently contains the **Frontend UI/UX implementation** and utilizes mock data to demonstrate the platform's core logic.

---

## ✨ Key Features

### 📂 Document Lifecycle
A structured submission process to ensure quality:
- **Draft:** Save and edit your notes before submitting.
- **Under Review:** Real-time analysis by our (mock) AI scoring engine.
- **Published:** Once verified, notes are visible to the community and available for download.

### 🤖 Smart Moderation (Admin Panel)
A dedicated administrative interface for platform health:
- **AI Quality Scoring:** Automated quality checks with adjustable thresholds.
- **Content Moderation:** Flagged documents queue and report management.
- **System Control:** Manage user accounts and system-wide settings.

### 🏅 Gamification & Profile
Building an active academic community through rewards:
- **CampusCoins:** Earn coins for every high-quality contribution.
- **Virtual Ranks:** Move up from Silver to Platinum based on your contributions.
- **Profile Analytics:** Track your downloads, likes, and average AI performance.

### 🔍 Discovery
- **Smart Search:** Find notes by course code, faculty, or title.
- **Dynamic Filtering:** Filter resources by Faculty and Department.
- **Top Contributors:** Leaderboard for the university's most helpful students.

---

## 🛠 Tech Stack

- **Framework:** [React 18](https://reactjs.org/) with [Vite](https://vitejs.dev/)
- **Language:** TypeScript
- **Styling:** Tailwind CSS (Modern, Responsive, Dark Mode support)
- **Icons:** Lucide React
- **Routing:** React Router DOM
